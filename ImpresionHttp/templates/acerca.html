Mi Aplicación Flask

Esta es una aplicación web simple creada con Flask para gestionar tareas.

Características:
  • Agregar nuevas tareas
  • Marcar tareas como completadas
  • Eliminar tareas
  • Ver estadísticas de tareas
Tecnologías Utilizadas
  • Python 3.x
  • Flask Framework
  • HTML5
  • CSS3
  • Bootstrap 5
  • JavaScript
Funcionalidades
  • Servidor web local
  • Almacenamiento en memoria
  • Diseño responsivo
  • Mensajes flash
  • Validación de formularios
  • Estadísticas en tiempo real
¿Por qué Flask?

Flask es un microframework web ligero y flexible para Python. Es perfecto para:

  • Aplicaciones web simples y rápidas
  • Prototipado rápido
  • Aprendizaje de desarrollo web
  • APIs RESTful
  • Proyectos pequeños y medianos
